Curry House Online Ordering System

Project Overview

A comprehensive online ordering platform designed specifically for Curry House restaurant located in South Canteen. The system streamlines the ordering process, improves customer experience, and provides restaurant owners with efficient order management tools.

Key Features

Interactive menu with real-time availability updates
Secure user authentication and registration system
Order tracking with real-time status updates
Payment gateway integration (Stripe/PayPal)

Technical Implementation

Built using React.js for the frontend with Redux for state management. Backend powered by Node.js and Express with MongoDB database. Implements RESTful API architecture and follows responsive design principles.

React.js Node.js MongoDB Express.js Redux Stripe API

Project Challenges & Solutions

One major challenge was handling concurrent orders during peak hours. Implemented a queue-based order processing system that manages load efficiently. Another challenge was real-time inventory updates - solved using WebSocket connections for instant updates.

Results & Impact

The system reduced order processing time by 40% and increased customer satisfaction by 60%. Restaurant reported 30% increase in online orders within first month of implementation.

Smart IoT Sensor Board

Project Overview

A custom-designed 4-layer PCB for environmental monitoring applications, integrating multiple sensors and wireless communication capabilities for IoT deployments.

Key Features

4-layer PCB design with optimized power distribution
ESP32 microcontroller with Wi-Fi/Bluetooth
Multiple sensor interfaces (I2C, SPI, UART, ADC)
Low-power design with sleep modes

Technical Specifications

Board dimensions: 50mm x 50mm. Operating voltage: 3.3V with 5V input capability. Includes temperature, humidity, pressure, and air quality sensors. Supports OTA (Over-the-Air) firmware updates.

Altium Designer ESP32 I2C/SPI Sensor Fusion PCB Fabrication IoT Protocols

Design Challenges

Major challenges included managing heat dissipation from the ESP32, minimizing EMI for accurate sensor readings, and optimizing board space for all components while maintaining signal integrity.

Testing & Validation

Conducted extensive testing including signal integrity analysis, thermal imaging, and environmental stress testing. Achieved 99.8% sensor accuracy and 6-month battery life in low-power mode.

Smart Hydroponic Monitoring System

Project Overview

An IoT-based automated hydroponic farming system that monitors and controls plant growth parameters in real-time, optimizing yield and resource usage.

Key Features

Real-time pH, EC, and nutrient level monitoring
Automated nutrient dosing with peristaltic pumps
Mobile app for remote monitoring and control
Climate control (temperature, humidity, light cycles)

System Architecture

Uses Raspberry Pi as central controller with Arduino nodes for sensor data collection. Cloud backend for data storage and analysis. Mobile app built with React Native for cross-platform compatibility.

Raspberry Pi Arduino React Native Node.js MQTT Sensor Networks

Control Algorithms

Implemented PID controllers for precise nutrient dosing and climate control. Machine learning algorithms analyze plant growth patterns to optimize nutrient mixtures and light cycles.

Results & Benefits

Increased crop yield by 40% compared to traditional methods. Reduced water usage by 70% and nutrient waste by 60%. System can be scaled from small home setups to commercial farms.

Project Overview

Key Features

Energy consumption monitoring and optimization
Security system with motion detection and cameras
Voice control with Google Assistant and Alexa
Automated lighting and climate control

System Architecture

Home Assistant Raspberry Pi ESP32 Z-Wave/Zigbee Node-RED MQTT
