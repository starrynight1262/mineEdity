Mala Hotpot Store Website

Project Overview

A complete HTML/CSS/JavaScript website for a Mala Hotpot restaurant, featuring an interactive menu, online ordering system, and responsive design.

Project Images

Key Features

Interactive menu with spice level customization (微辣, 中辣, 大辣)
Real-time price calculation as customers select ingredients
Mobile-responsive design that works on all devices
Order summary and printable receipt generation

Technical Implementation

Pure HTML5, CSS3, and vanilla JavaScript implementation. No frameworks used to ensure fast loading times and compatibility. CSS Grid and Flexbox for responsive layouts.

HTML5 CSS3 JavaScript CSS Grid Flexbox Responsive Design

Implementation Details

Design Challenges

Major challenge was creating an intuitive interface for complex ingredient selection. Implemented a category-based selection system with visual feedback. Another challenge was ensuring the website loaded quickly on mobile devices - achieved by optimizing images and minimizing JavaScript.

M5 Stack Plant Care System

Project Overview

An automated plant monitoring and care system using M5 Stack devices with block-based programming (UIFlow) to monitor soil conditions and automate watering.

Project Images

Key Features

Real-time soil moisture monitoring with capacitive sensors
Temperature and humidity tracking using DHT11 sensor
Automated watering system with peristaltic pump control
Block-based programming using M5 Stack UIFlow

System Architecture

M5 Stack Core2 as the main controller with Grove soil moisture sensor and DHT11 temperature/humidity sensor. Water pump controlled via relay module. Programming done using UIFlow's block-based interface with custom logic blocks.

M5 Stack Core2 UIFlow Block Programming Soil Moisture Sensor DHT11 Relay Control

Programming Logic

Implemented using UIFlow's visual programming blocks: When soil moisture drops below 30%, activate pump for 5 seconds. When temperature exceeds 35°C, send alert notification. Data logged to M5 Stack SD card for analysis.

Results & Benefits

Maintained optimal soil moisture levels (±5% accuracy). Reduced water usage by 40% compared to manual watering. Plants showed 25% better growth rate with consistent conditions. System can run autonomously for 2 weeks on battery.

BeagleBone Embedded Dashboard

Project Overview

Embedded IoT system using BeagleBone Black running Python Flask server with HTML dashboard for real-time sensor monitoring and control through web interface.

Project Images

Key Features

BeagleBone Black as embedded Linux server
Remote access via Putty SSH terminal
Python Flask backend for web server
Real-time HTML dashboard with Chart.js visualizations

System Architecture

BeagleBone Black running Debian Linux with Python 3. Flask web server handles HTTP requests and serves HTML/CSS/JS dashboard. GPIO pins read sensor data (DHT22, light sensor). Data updates in real-time via AJAX calls.

BeagleBone Black Python Flask HTML/CSS/JS Chart.js Putty SSH GPIO Control

Implementation Details

System accessed via Putty SSH terminal for initial setup and debugging. Python script runs continuously, reading sensors every 10 seconds. Flask server provides REST API endpoints for frontend. Dashboard uses AJAX to poll data every 5 seconds. Control buttons trigger GPIO outputs for relays/LEDs.

Technical Challenges

Challenge 1: Setting up Flask to run on boot - solved with systemd service. Challenge 2: Real-time updates without page refresh - solved with JavaScript setInterval() and AJAX. Challenge 3: GPIO permission issues - solved with udev rules configuration.
